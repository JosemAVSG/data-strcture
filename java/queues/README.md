# Colas

## Problemas

1. Implementa una clase Cola (Queue) con métodos enqueue, dequeue, peek y is_empty.
2. Implementa una cola usando dos pilas.
3. Implementa una función que simule una cola de impresión con prioridades.
4. Implementa una función que invierta los primeros k elementos de una cola.
5. Implementa una función que genere números binarios usando una cola.
