# Grafos

## Problemas

1. Implementa una clase Grafo con representación de lista de adyacencia y métodos para agregar vértices y aristas.
2. Implementa una función que realice un recorrido BFS (Breadth-First Search) en un grafo.
3. Implementa una función que realice un recorrido DFS (Depth-First Search) en un grafo.
4. Implementa una función que detecte ciclos en un grafo no dirigido.
5. Implementa el algoritmo de Dijkstra para encontrar el camino más corto en un grafo con pesos.
