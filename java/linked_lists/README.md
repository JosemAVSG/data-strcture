# Listas Enlazadas

## Problemas

1. Implementa una clase para una lista enlazada simple con métodos para insertar un nodo al inicio, al final, y eliminar un nodo dado su valor.
2. Implementa una función que invierta una lista enlazada.
3. Implementa una función que detecte si hay un ciclo en una lista enlazada.
4. Implementa una función que encuentre el nodo medio de una lista enlazada.
5. Implementa una función que fusione dos listas enlazadas ordenadas en una sola lista ordenada.
