# Tries

## Problemas

1. Implementa una clase Trie (prefijo) con métodos para insertar y buscar palabras.
2. Implementa una función que cuente el número de palabras que empiezan con un prefijo dado en un trie.
3. Implementa una función que elimine una palabra de un trie.
4. Implementa una función que encuentre todas las palabras que empiezan con un prefijo dado.
5. Implementa un trie para almacenar números de teléfono y buscar contactos por prefijo.
