# Arreglos

## Problemas

1. Implementa una función que encuentre el elemento máximo en un arreglo.
2. Implementa una función que calcule la suma de todos los elementos en un arreglo.
3. Implementa una función que busque un elemento específico en un arreglo y devuelva su índice (o -1 si no existe).
4. Implementa una función que ordene un arreglo en orden ascendente usando un algoritmo de ordenamiento simple.
5. Implementa una función que elimine los elementos duplicados de un arreglo.
