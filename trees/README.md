# Árboles

## Problemas

1. Implementa una clase para un árbol binario con métodos para inserción y recorrido (inorden, preorden, postorden).
2. Implementa una función que calcule la altura de un árbol binario.
3. Implementa una función que verifique si un árbol binario es un árbol de búsqueda binario (BST).
4. Implementa una función que encuentre el ancestro común más bajo (LCA) de dos nodos en un árbol binario.
5. Implementa una función que convierta un arreglo en un árbol binario de búsqueda balanceado.
