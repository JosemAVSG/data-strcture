# Pilas

## Problemas

1. Implementa una clase Pila (Stack) con métodos push, pop, peek y is_empty.
2. Implementa una función que use una pila para verificar si una expresión de paréntesis está balanceada.
3. Implementa una función que invierta una cadena usando una pila.
4. Implementa una función que evalúe una expresión postfix usando una pila.
5. Implementa una función que simule el comportamiento de una pila con dos colas.
