# 🏦 Banco "Trust&Safe" - Colas (Queues)

## Contexto
Trabajas en un banco. Las colas representan la atención a clientes.

### Ejercicio 1: Cola de Clientes 👥
Implementa una cola para atender clientes en orden de llegada. El primero en llegar es el primero en ser atendido (FIFO).

### Ejercicio 2: Cola con Dos Pilas 🔧
Ejercicio teórico: implementar una cola usando dos pilas.

### Ejercicio 3: Cola de Préstamos con Prioridad 💰
Los préstamos tienen prioridad. Los clientes con mayor urgencia (mayor número) se atienden primero.

### Ejercicio 4: Invertir Primeros K Clientes 🔄
El gerente quiere invertir el orden de los primeros K clientes en la fila para cambiar el orden de atención.

### Ejercicio 5: Generar Tickets de Turno 🎫
Sistema automático de turnos: 1, 10, 11, 100, 101... (secuencia binaria).
