# 🏭 Fabrica de Electronics - Departamento de Arrays

## Contexto
Trabajas en una fabrica de electrónicos. Tu objetivo es resolver problemas logísticos usando arrays.

### Ejercicio 1: Control de Temperatura del Horno 🔥
En la línea de soldadura, los hornos deben mantener una temperatura óptima entre 180°C y 220°C. Tienes un array con las lecturas de temperatura de los últimos 10 ciclos. Necesitas encontrar la temperatura máxima registrada para verificar que el horno no se sobrecalentó.

**Contexto**: "El supervisor necesita un reporte de la temperatura máxima para el registro de calidad. Si supera los 230°C, hay que detener la línea."

### Ejercicio 2: Inventario de Componentes 📦
La fábrica recibió un envío de componentes. Tienes un array con las cantidades de cada tipo de componente. Calcula el total de componentes recibidos para hacer el inventario.

**Contexto**: "El departamento de contabilidad necesita saber el total de componentes para calcular el costo del envío."

### Ejercicio 3: Buscar Componente Específico 🔍
En el almacén hay miles de componentes. Necesitas buscar si existe un componente con código específico para saber si está disponible.

**Contexto**: "Un cliente llama preguntando si tenemos el componente SKU-757 para un pedido urgente."

### Ejercicio 4: Ordenar Pedidos por Prioridad 📋
Los pedidos de producción llegan en orden aleatorio. Necesitas ordenarlos por fecha de entrega para priorizar los más urgentes.

**Contexto**: "El gerente de producción necesita ver los pedidos ordenados por urgencia para asignar recursos."

### Ejercicio 5: Eliminar Duplicados en Lista de Proveedores 🏭
Tienes una lista de proveedores con códigos repetidos. Necesitas una lista limpia sin duplicados para enviar órdenes de compra.

**Contexto**: "No podemos enviar múltiples órdenes al mismo proveedor, necesitamos una lista única."
