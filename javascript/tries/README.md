# 🔍 Motor de Búsqueda "QuickSearch" - Tries

## Contexto
Trabajas en QuickSearch, un motor de búsqueda. Los tries son perfectos para autocompletado.

### Ejercicio 1: Trie de Búsquedas 🔤
Implementa un Trie para almacenar búsquedas recientes de usuarios.

### Ejercicio 2: Contar Palabras con Prefijo 📊
Cuenta cuántas búsquedas empiezan con un prefijo específico.

### Ejercicio 3: Eliminar Búsqueda Reciente 🗑️
Permite eliminar una búsqueda del historial.

### Ejercicio 4: Sugerencias de Búsqueda 💡
Dado un prefijo, sugiere todas las búsquedas que empiezan con ese prefijo.

### Ejercicio 5: Autocompletado de Contactos 📱
Simula el autocompletado de contactos al escribir un mensaje.
