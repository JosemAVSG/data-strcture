# 🌳 Empresa GeneTech - Árboles Genealógicos

## Contexto
Trabajas en GeneTech, una empresa que mapea árboles genealógicos familiares.

### Ejercicio 1: Árbol Familiar con Recorridos 👨‍👩‍👧
Implementa un árbol genealógico donde cada nodo es una persona. Recorre en diferentes órdenes (preorden, inorden, postorden).

### Ejercicio 2: Altura del Árbol Familiar 📏
Calcula cuántas generaciones tiene una familia (altura del árbol).

### Ejercicio 3: Validar Árbol de Búsqueda 🔍
Verifica si el árbol genealógico cumple las reglas de un BST (todos los hijos menores al padre, mayores al abuelo, etc.).

### Ejercicio 4: Ancestro Común Más Bajo 👪
Encuentra el ancestro común más bajo (LCA) de dos personas en el árbol familiar.

### Ejercicio 5: Arreglo a Árbol Balanceado ⚖️
Convierte un arreglo de personas ordenadas por edad a un árbol balanceado.
