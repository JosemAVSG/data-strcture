# 🗄️ Base de Datos "DataStore" - Tablas Hash

## Contexto
Trabajas en DataStore, una base de datos NoSQL. Las tablas hash son fundamentales.

### Ejercicio 1: Tabla Hash de Usuarios 👤
Implementa una tabla hash simple para almacenar usuarios por ID.

### Ejercicio 2: Manejar Colisiones con Chaining 🔗
Maneja colisiones (dos keys diferentes con mismo hash) usando listas enlazadas.

### Ejercicio 3: Manejar Colisiones con Linear Probing 📍
Maneja colisiones buscando el siguiente espacio libre.

### Ejercicio 4: Contar Frecuencia de Búsquedas 🔎
Analiza las búsquedas más frecuentes de los usuarios.

### Ejercicio 5: Primer Elemento No Repetido ✨
