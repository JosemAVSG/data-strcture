# 🍕 Pizzeria "Mama Mia" - Pilas (Stacks)

## Contexto
Trabajas en una pizzeria. Las pilas representan la cocina y pedidos.

### Ejercicio 1: Pila de Pizzeros 👨‍🍳
Implementa una pila para gestionar los pedidos de pizzas. El último pedido en entrar debe ser el primero en prepararse.

### Ejercicio 2: Validar Pedidos de Paréntesis 🍕
El sistema de pedidos tiene expresiones como "((HAMBURGUESA + PAPAS) * REFRESCO)". Necesitas validar que los paréntesis estén balanceados.

### Ejercicio 3: Invertir Nombres de Pedidos 🔄
Para hacer una broma, el chef quiere ver los nombres de pedidos invertidos. Usa una pila para esto.

### Ejercicio 4: Calculadora de Pedidos 🧮
El sistema evalúa expresiones en notación postfija: "2 3 +" significa "2 + 3 = 5". Calcula el total de un pedido.

### Ejercicio 5: Pila con Dos Colas 🍽️
Simula una pila usando dos colas (ejercicio teórico de estructuras).
