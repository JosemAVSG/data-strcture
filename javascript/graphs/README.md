# 🌐 Red Social "ConnectAll" - Grafos

## Contexto
Trabajas en ConnectAll, una red social. Los grafos representan conexiones entre usuarios.

### Ejercicio 1: Grafo de Amigos 👥
Implementa un grafo donde cada usuario puede seguir a otros. Usa lista de adyacencia.

### Ejercicio 2: BFS - Encontrar Conexiones 🔍
Usa BFS para encontrar a alguien en la red de amigos de amigos.

### Ejercicio 3: DFS - Explorar Perfil 👤
Usa DFS para explorar el perfil de un usuario y sus seguidores.

### Ejercicio 4: Detectar Ciclos en Following 🔄
Detecta si hay un ciclo en las relaciones de follows (el usuario A sigue a B, B sigue a C, C sigue a A).

### Ejercicio 5: Dijkstra - Encontrar Camino Más Corto ⭐
Encuentra el camino más corto entre dos usuarios para sugerir amigos en común.
