# 🚚 Empresa de Entregas "RapidBox" - Listas Enlazadas

## Contexto
Trabajas en una empresa de mensajería. Las listas enlazadas representan las rutas de entrega.

### Ejercicio 1: Sistema de Seguimiento de Paquetes 📦
Cada paquete tiene un ID y necesita ser agregado al sistema de seguimiento. El sistema debe permitir agregar paquetes al inicio (nuevos pedidos urgentes) y al final (entregas regulares), además de eliminar paquetes entregados.

### Ejercicio 2: Invertir Ruta de Entregas 🔄
El GPS falló y necesitas invertir el orden de las entregas en una ruta para optimizarla.

### Ejercicio 3: Detector de Ciclos en Rutas 🚨
Hay un bug en el sistema que crea rutas circulares. Necesitas detectar si una ruta tiene un ciclo infinito para evitar que los repartidores queden atrapados.

### Ejercicio 4: Punto Medio de la Ruta 📍
Necesitas encontrar el punto medio de una ruta para asignar un segundo repartidor que ayude en la segunda mitad.

### Ejercicio 5: Fusionar Rutas Ordenadas 🛣️
Tienes dos rutas de entrega ya ordenadas por distancia desde el almacén. Fusionarlas en una sola ruta optimizada.
