# Tablas Hash

## Problemas

1. Implementa una clase TablaHash simple con métodos para insertar, buscar y eliminar elementos.
2. Implementa una función que maneje colisiones usando encadenamiento (chaining).
3. Implementa una función que maneje colisiones usando direccionamiento abierto (open addressing).
4. Implementa una función que calcule la frecuencia de elementos en un arreglo usando una tabla hash.
5. Implementa una función que encuentre el primer elemento que no se repite en un arreglo usando una tabla hash.
